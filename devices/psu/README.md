# PSU – Power Supply Unit
ספק כוח מעבדתי משמש להזנת רכיבים חשמליים במתח וזרם נשלטים.  
מטרתו במערכת: לאפשר שליטה מרחוק על מתח, זרם, והפעלה/כיבוי יציאה.

## נתונים (read):
- voltage (V)
- current (A)
- temp (°C)
- output (bool)

## פעולות (set):
- set_voltage (V)
- set_current_limit (A)
- toggle_output (on/off)
- power_cycle

## יחידות
- voltage: V
- current: A
- temp: °C
